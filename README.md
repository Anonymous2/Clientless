Clientless
=============

[![Build Status](https://drone.io/github.com/Dehravor/Clientless/status.png)](https://drone.io/github.com/Dehravor/Clientless/latest)

Clientless is a World of Warcraft client emulator for 4.3.4 15595 Cataclysm.

Requirements
-------

 + CMake ≥ 2.6
 + OpenSSL ≥ 0.9.7
 + C++11 Compiler
